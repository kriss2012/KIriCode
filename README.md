## 🌟 Overview

**KiriCode** is a modern web-based code editor designed for developers, students, and coding enthusiasts. It provides an intuitive coding environment with syntax highlighting, file management, and real-time editing capabilities.

The platform aims to simplify coding workflows while offering a clean and responsive user experience.

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/kriss2012/KIriCode.git
```

### Navigate to Project

```bash
cd KIriCode
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm start
```

Application runs at:

```text
http://localhost:3000
```

---

## ⚙️ Environment Variables

Create a `.env` file:

```env
PORT=5000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key

API_KEY=your_api_key
```
